## Install

Use `chezmoi` to initate a new machine.

```bash
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply git@github.com:nickpellant/dotfiles.git
```

See [chezmoi](https://chezmoi.io/) documentation for further instructions.
